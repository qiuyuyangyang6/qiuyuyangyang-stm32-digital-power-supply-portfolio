# STM32 Digital Power Supply Portfolio

## 项目简介

本仓库用于展示我的个人项目：

**基于 STM32G474 的四开关同步整流 Buck-Boost 数字电源复刻与调试项目**

项目围绕 `主功率拓扑分析`、`采样调理电路设计`、`HRTIM + ADC + DMA 高速控制链路`、`CV/CC 双环控制`、`Type-C PD 输入`、`OLED 人机交互` 和 `整机调试验证` 展开，适合作为简历中的个人项目展示链接。

## 项目定位

- 项目类型：个人硬件/嵌入式项目
- 技术方向：数字电源、STM32、功率电子、电源控制
- 项目角色：独立完成复刻分析、原理学习、调试验证与项目资料整理

## 核心内容

### 1. 电源系统设计

- 四开关同步整流 Buck-Boost 主功率拓扑
- 输入范围 `12V-48V`
- 输出范围 `0.5V-48V / 0-10A`
- 最高功率 `450W`
- Buck / Boost / Mix 三工作模式切换

### 2. 控制与采样方案

- 基于 `STM32G474` 搭建数字控制核心
- 使用 `HRTIM` 产生高分辨率互补 PWM，驱动同步整流 MOS
- 使用 `ADC + DMA` 对 `Vin / Iin / Vout / Iout` 进行实时采样
- 采样前端采用：
  - 电压分压采样
  - `5mOhm` 分流电阻采样
  - `GS8558` 差分放大
  - `REF3033` 基准源提高 ADC 精度
- 软件实现 `CV / CC` 双环闭环控制与保护逻辑

### 3. 辅助功能

- `CH224K` 实现 Type-C PD 输入协商
- `OLED + 旋转编码器 + 按键` 完成人机交互
- `W25Q64` 存储参数配置
- 支持过压、过流、短路、过温保护
- 支持串口上位机数据观测
